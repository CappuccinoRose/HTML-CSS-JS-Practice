# HTML 语法知识小练习

本仓库包含 9 个精心设计的 HTML/CSS/JS 练习案例，旨在通过实战代码演示前端开发中的核心概念与现代特性。内容涵盖基础布局、CSS 进阶特效、原生 JavaScript 交互以及 HTML5 新增 API 的应用。

## 📂 案例目录

### 第一部分：基础结构与样式

| 文件名 | 案例主题 | 核心知识点 |
| :--- | :--- | :--- |
| `01.html` | **基础结构与样式入门** | HTML 文档骨架 (`head`, `body`)、内联样式 vs 内部样式表、基础布局概念。 |
| `02.html` | **CSS 盒子阴影特效** | `box-shadow` 语法详解（偏移、模糊、扩展）、卡片UI设计、霓虹灯特效实现。 |
| `03.html` | **浮动与文本环绕** | CSS `float` 属性、图文混排技巧、`shape-outside` 高级形状排版。 |

### 第二部分：现代布局与交互组件

| 文件名 | 案例主题 | 核心知识点 |
| :--- | :--- | :--- |
| `04.html` | **详情展开/收起** | 原生 JS 事件监听 (`addEventListener`)、`classlist.toggle` 状态切换、CSS 过渡动画。 |
| `05.html` | **Popover 弹出菜单** | HTML5 最新 `popover` 属性、CSS 锚点定位、`::backdrop` 伪类、无 JS 依赖的交互组件。 |
| `06.html` | **模态对话框** | `<dialog>` 元素、`showModal()` 方法、表单提交关闭对话框 (`method="dialog"`)、遮罩层样式。 |
| `07.html` | **原生拖拽功能** | Drag and Drop API (`dragstart`, `dragover`, `drop`)、`dataTransfer` 数据传递、拖拽状态样式控制。 |

### 第三部分：表单控件与响应式技术

| 文件名 | 案例主题 | 核心知识点 |
| :--- | :--- | :--- |
| `08.html` | **系统控制面板** | `<progress>` 进度条、`<meter>` 度量衡、`<datalist>` 自动补全、CSS 渐变与美化原生控件。 |
| `09.html` | **响应式图片切换** | `<picture>` 元素、`srcset` 与 `sizes` 属性、`media` 媒体查询适配不同屏幕宽度。 |

## 🚀 快速开始

1.  **克隆或下载** 本仓库到本地。
2.  直接在浏览器中打开任意 `.html` 文件即可查看效果。
3.  建议使用 Chrome、Edge 或 Firefox 等现代浏览器以获得最佳体验（特别是 `05.html` 和 `06.html` 使用了较新的 HTML5 特性）。

## 💡 学习建议

*   **基础阶段**：先阅读 `01.html` 和 `02.html`，理解 HTML 结构与 CSS 盒模型。
*   **进阶布局**：通过 `03.html` 学习浮动布局的历史与进阶玩法。
*   **JS 交互**：研究 `04.html` 和 `07.html`，掌握原生 JavaScript 如何操作 DOM 和处理事件。
*   **现代特性**：重点学习 `05.html` (Popover) 和 `06.html` (Dialog)，它们代表了现代 Web 开发的原生组件化趋势。
*   **工程化思维**：通过 `09.html` 理解前端性能优化中“响应式图片”的重要性。

## 🛠️ 技术栈

*   **HTML5**: 语义化标签、Web Components 原生属性。
*   **CSS3**: Flexbox 布局、Transition 动画、Shadow/Shape 高级特性。
*   **JavaScript (ES6+)**: 原生事件流、DOM 操作。
